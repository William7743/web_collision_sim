## 🔧 开发指南

### 添加新形状
在 `js/main.js` 中添加方法：
```javascript
addNewShape() {
    // 创建 Cannon.js 刚体
    const body = new CANNON.Body({...});
    this.world.addBody(body);

    // 创建 Three.js 网格
    const mesh = new THREE.Mesh({...});
    this.scene.add(mesh);

    // 同步
    this.objects.push({ body, mesh });
}
```

### 添加新场景
参考 `createStack()` 方法，创建场景函数并添加到 HTML 按钮。

### 运行测试
```bash
# 单元测试
npm run test:unit

# E2E 测试
npm run test:e2e

# 所有测试
npm test
```

## 🐛 已知问题

### 1. Node.js 环境限制
- **问题**: `CANNON.Compound` 在 Node.js 中不可用
- **影响**: 单元测试中复合形状测试被跳过
- **解决方案**: 浏览器环境中正常工作
- **状态**: 已处理，测试标记为 SKIP

### 2. 大量物体时性能下降
- **问题**: 物体超过 300 个时 FPS 下降
- **影响**: 极端场景下可能卡顿
- **解决方案**: 限制物体数量或使用对象池
- **状态**: 已知，不影响正常使用
