📍 访问地址

编辑器地址： http://localhost:5173/editor.html

主页地址： http://localhost:5173/index.html

✅ 服务器正在运行

🆕 新增功能测试

1. 速度设置功能

测试步骤：
  1. 打开编辑器
  2. 点击"📦 盒子"添加一个刚体
  3. 选中刚体后，查看右侧面板
  4. 应该看到"初始速度 (m/s)"和"初始角速度 (rad/s)"两个新区域
  5. 每个区域有X/Y/Z三个输入框

ℹ️ 如果看不到速度设置，请按F12打开控制台查看错误

2. 约束设置改进

测试步骤：
  1. 添加2个盒子
  2. 在左侧面板找到"添加约束"区域
  3. 应该看到"已选择的刚体:"显示区域
  4. 应该看到"⚡ 创建约束"按钮
  5. 按住Ctrl点击盒子（应该变绿）

ℹ️ 按住Ctrl点击刚体时会看到绿色高亮和"✓ 已选择用于约束"提示

🔧 故障排除

问题1：页面空白

解决方法：

  • 按F12打开开发者工具
  • 查看Console标签页是否有错误
  • 检查Network标签页，确认editor.js是否成功加载

问题2：看不到新功能

解决方法：

  • 清除浏览器缓存（Ctrl+Shift+Delete）
  • 硬刷新页面（Ctrl+F5或Cmd+Shift+R）
  • 确认访问的是editor.html而不是index.html

问题3：刚体无法选中

解决方法：

  • 确保在编辑模式下（左上角显示"编辑模式"）
  • 点击刚体应该会变成蓝色高亮
  • 右侧面板应该显示参数编辑界面

问题4：Ctrl+点击不生效

解决方法：

  • 确保按住Ctrl键的同时点击
  • Mac用户请使用Cmd键代替Ctrl
  • 在刚体列表中点击，而不是在3D视图中点击

💻 使用示例

示例1：测试速度设置

  1. 添加一个球体
  2. 选中球体
  3. 设置：位置Y=5, 速度X=5, 速度Y=10
  4. 点击"▶ 运行模拟"
  5. 应该看到球体向斜上方飞出

示例2：测试约束创建

  1. 添加2个盒子
  2. 在左侧刚体列表中，按住Ctrl点击第1个盒子
  3. 继续按住Ctrl点击第2个盒子
  4. 两个盒子都应该变成绿色
  5. 点击"⚡ 创建约束"按钮
  6. 点击"▶ 运行模拟"查看效果
