## 🎯 项目简介

这是一个基于 **cannon-es** 物理引擎和 **Three.js** 渲染引擎的刚体碰撞仿真 Web 应用。支持多种刚体形状、复合形状、约束系统，以及丰富的预设场景。

### 技术栈
- **物理引擎**: cannon-es (纯 JavaScript)
- **渲染引擎**: Three.js
- **构建工具**: Vite
- **测试框架**: Puppeteer (E2E)

## ✨ 功能特性

### 基础形状
- ✅ 盒子 (Box)
- ✅ 球体 (Sphere)
- ✅ 圆柱 (Cylinder)
- ✅ 平面 (Plane)

### 复杂功能
- ✅ 复合形状 - 组合多个基础形状
- ✅ 铰链约束 (Hinge Constraint)
- ✅ 点对点约束 (Point-to-Point)
- ✅ 鼠标拖拽交互
- ✅ 实时物理模拟 (60 FPS)

### 预设场景
- ⚖️ 钟摆链 - 5个铰链链接的物理钟摆
- 🔗 铰链盒子 - 4个用铰链连接的彩色盒子
- 🚗 车辆模型 - 带车轮的复合车身
- 📚 堆叠测试 - 24个盒子的稳定堆叠
- 🔺 金字塔 - 15个盒子的金字塔结构
- 🌧️ 球体雨 - 30个随机大小的球体掉落

## 🎨 自定义编辑器

### 简介

除了预设场景，我们还提供了一个功能强大的**可视化编辑器**，让你能够：

- ✅ 自由添加各种刚体（盒子、球体、圆柱、复合体）
- ✅ 精确设置物理参数（质量、摩擦力、弹性等）
- ✅ 使用多种约束连接刚体（铰链、点对点、距离、固定）
- ✅ 实时编辑和调整参数
- ✅ 一键运行物理模拟
- ✅ 保存和加载自定义场景

### 访问编辑器

**方式1：从主页进入**
1. 在主页面左下角找到"🎨 打开编辑器"按钮
2. 点击按钮进入编辑器

**方式2：直接访问**
```
http://localhost:5173/editor.html
```

### 编辑器核心功能

#### 1. 添加刚体
在左侧面板点击对应按钮：
- 📦 **盒子** - 长方体刚体
- 🔮 **球体** - 球形刚体
- 🥫 **圆柱** - 圆柱形刚体
- 🧩 **复合体** - 复杂组合刚体

#### 2. 编辑参数
选中刚体后，右侧面板可编辑：
- **名称** - 刚体显示名称
- **质量** - 0=静态，>0=动态
- **摩擦力** - 0=光滑，1=高摩擦
- **弹性** - 0=不弹，1=超级弹
- **位置** - X/Y/Z 坐标
- **尺寸** - 根据类型不同而不同

#### 3. 添加约束
1. 在左侧选择约束类型（铰链/点对点/距离/固定）
2. 按住 `Ctrl` 键
3. 依次点击两个刚体
4. 约束自动创建

#### 4. 运行模拟
- **▶ 运行模拟** - 启动物理引擎
- **⏹ 停止** - 暂停模拟，返回编辑模式
- **↺ 重置位置** - 所有物体回到初始状态

#### 5. 场景管理
- **💾 保存场景** - 保存为JSON文件
- **📂 加载场景** - 从文件加载
- **🗑 清空** - 删除所有内容

## 💻 开发指南

### 添加新的形状

1. 在 `main.js` 中添加新方法：

```javascript
addNewShape() {
    const position = {
        x: Math.random() * 4 - 2,
        y: 5 + Math.random() * 5,
        z: Math.random() * 4 - 2
    };

    // Cannon.js 刚体
    const shape = new CANNON.YourShape(...);
    const body = new CANNON.Body({
        mass: 1,
        shape: shape,
        position: new CANNON.Vec3(position.x, position.y, position.z)
    });
    this.world.addBody(body);

    // Three.js 网格
    const geometry = new THREE.YourGeometry(...);
    const material = new THREE.MeshPhongMaterial({
        color: Math.random() * 0xffffff
    });
    const mesh = new THREE.Mesh(geometry, material);
    mesh.position.set(position.x, position.y, position.z);
    mesh.castShadow = true;
    this.scene.add(mesh);

    this.objects.push({ body, mesh });
    this.updateObjectCount();
}
```

2. 在 `index.html` 中添加按钮：


## 🐛 常见问题

### Q: 物体穿过地面？
A: 确保物体质量设置合理，过重的物体可能穿透。可以增加求解器迭代次数：
```javascript
world.solver.iterations = 20;  // 默认是 10
```

### Q: FPS 过低？
A: 减少物体数量，或降低物体网格的面数。

### Q: 鼠标拖拽不工作？
A: 确保点击的是物体而不是背景。左键用于拖拽物体，右键用于旋转视角。

### Q: 如何保存场景？
A: 当前版本不支持场景保存。可以手动记录物体的位置和类型，然后在代码中重现。

## 📊 性能优化建议

1. **限制物体数量** - 建议不超过 200 个动态物体
2. **使用简单形状** - 盒子和球体比复合形状性能更好
3. **减少阴影** - 对于大量物体，可以关闭 `castShadow`
4. **调整求解器** - 减少迭代次数可以提高性能但降低精度
