# 🎉 新功能快速开始

## ✨ 已添加的功能

### 1. ⚖️ 重力设置
- X/Y/Z轴独立控制
- 实时生效
- 模拟不同星球重力

### 2. 📦 边界删除系统
- 设置活动空间
- 超出边界自动删除
- 红色边界框可视化

---

## 🚀 立即体验（30秒）

### 第1步：打开编辑器
```
http://localhost:5173/editor.html
```

### 第2步：找到新功能
在左侧面板找到"🌍 世界设置"

### 第3步：测试重力
```
1. 添加一个球体
2. 修改重力 Y = 0（零重力）
3. 运行模拟
4. 球体应该悬浮不动
```

### 第4步：测试边界
```
1. 勾选"启用边界删除"
2. 设置Y轴最小值 = -3
3. 运行模拟
4. 球体掉出边界后被删除
5. 底部显示"已删除: 1"
```

## ⚙️ 关键参数

### 重力常用值
- 地球: Y = -9.82
- 月球: Y = -1.62
- 火星: Y = -3.71
- 零重力: Y = 0
- 反向: Y = 9.82

### 边界推荐值
- 小场景: X/Y/Z ±10
- 中场景: X/Y/Z ±20
- 大场景: X/Y/Z ±50

## ⚠️ 注意事项

1. **清除缓存**（重要！）
   ```
   Mac: ⌘ + Shift + R
   Windows: Ctrl + F5
   ```

2. **边界范围**
   ```
   确保 最小值 < 最大值
   例如：minX=-10 < maxX=10 ✓
   ```

3. **边界删除只在运行时生效**
   ```
   必须点击"▶ 运行模拟"
   编辑模式下不会删除
   ```
