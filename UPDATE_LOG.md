# 🎨 自定义碰撞编辑器 - 更新日志

## v1.1.0 (2026-01-08)

### ✨ 新增功能

#### 1. 初始速度设置
- ✅ **线速度 (Velocity)** - 设置刚体在X/Y/Z方向的初始速度（单位：米/秒）
- ✅ **角速度 (Angular Velocity)** - 设置刚体的初始旋转速度（单位：弧度/秒）
- 📍 位置：右侧参数面板 → "初始速度" 和 "初始角速度" 区域

**使用场景：**
- 创建抛射物（设置初始Y轴速度向上）
- 创建旋转物体（设置角速度让物体自转）
- 模拟碰撞前的物体状态
- 创建更丰富的物理效果

**示例：**
```
创建一个球体，设置：
- 位置: (0, 5, 0)
- 初始速度: X=5, Y=10, Z=0
- 运行后会看到球体向斜上方飞出
```

#### 2. 改进的约束设置功能
- ✅ **可视化选择** - 已选择的刚体会显示绿色高亮
- ✅ **选择提示** - 刚体列表中显示"✓ 已选择用于约束"
- ✅ **创建按钮** - 新增"⚡ 创建约束"按钮，操作更明确
- ✅ **选择显示** - 实时显示已选择的刚体名称
- ✅ **智能提示** - 根据选择状态动态显示操作提示

**新约束创建流程：**
```
1. 添加2个或更多刚体
2. 按住 Ctrl 点击第一个刚体（变绿）
3. 继续按住 Ctrl 点击第二个刚体（也变绿）
4. 在左侧面板选择约束类型
5. 点击"⚡ 创建约束"按钮
6. 约束创建完成！
```

**约束类型说明：**
- 🔗 **铰链约束** - 像门铰链一样，可以绕轴旋转
- 📍 **点对点** - 两点连接，可以自由旋转
- 📏 **距离约束** - 保持两个刚体的固定距离
- 🔒 **固定约束** - 两个刚体完全固定在一起

### 📊 参数说明

| 参数 | 范围 | 单位 | 说明 |
|------|------|------|------|
| 线速度 X/Y/Z | -∞ ~ +∞ | m/s | 每秒移动的距离 |
| 角速度 X/Y/Z | -∞ ~ +∞ | rad/s | 每秒旋转的弧度 |
| 质量影响 | 0 = 静态，>0 = 动态 | kg | 质量越大，惯性越大 |
| 速度重置 | 自动 | - | 点击"重置位置"时恢复初始速度 |

### 🔧 技术细节

**速度保存机制：**
- 速度值保存在 `config.velocity` 和 `config.angularVelocity`
- 初始状态保存在 `initialTransform.velocity` 和 `initialTransform.angularVelocity`
- 点击"重置位置"时会恢复到初始速度
- 保存场景时速度参数也会被保存

**约束创建改进：**
- 使用 `selectedBodiesForConstraint` 数组存储选择
- `updateConstraintSelectionUI()` 实时更新显示
- CSS类 `constraint-selected` 提供视觉反馈
- 选择满2个后自动启用"创建约束"按钮
