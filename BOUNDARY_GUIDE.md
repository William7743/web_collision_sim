## 📚 功能详解

### 一、重力设置

#### 位置
左侧面板 → 世界设置 → 重力

#### 参数说明

| 参数 | 默认值 | 说明 | 使用场景 |
|------|--------|------|----------|
| **X轴** | 0 | 水平左右重力 | 侧向力场 |
| **Y轴** | -9.82 | 垂直重力 | 地球重力 |
| **Z轴** | 0 | 水平前后重力 | 深度力场 |

#### 常用重力预设

**地球重力（默认）**
```
X: 0
Y: -9.82
Z: 0
```

**月球重力**（地球的1/6）
```
X: 0
Y: -1.62
Z: 0
```

**零重力（太空）**
```
X: 0
Y: 0
Z: 0
```

**木星重力**（地球的2.5倍）
```
X: 0
Y: -24.79
Z: 0
```

**反向重力**
```
X: 0
Y: 9.82
Z: 0
物体会向上飞！
```

**倾斜重力**
```
X: 5
Y: -5
Z: 0
物体会向斜下方落
```

### 二、边界删除系统

#### 位置
左侧面板 → 世界设置 → 启用边界删除

#### 工作原理

```
┌─────────────────────────────┐
│  碰撞空间边界框（红色线框）   │
│                              │
│  ┌────────────────────────┐ │
│  │   物体在边界内 ✓       │ │
│  │                       │ │
│  └────────────────────────┘ │
│                              │
│  物体超出边界 → 自动删除      │
└─────────────────────────────┘
```

#### 使用步骤

**第1步：启用边界**
1. 勾选"启用边界删除"
2. 边界设置区域自动展开

**第2步：设置边界范围**
```
X轴范围: -10 到 10 （水平范围20米）
Y轴范围: -5 到 20  （高度范围25米）
Z轴范围: -10 到 10 （深度范围20米）
```

**第3步：可视化边界**
- 勾选"显示边界框"
- 3D视图中显示红色线框
- 表示物体的活动范围

**第4步：运行模拟**
- 点击"▶ 运行模拟"
- 物体超出边界自动删除
- 底部统计显示"已删除: N"

#### 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| **X轴最小值** | -10 | 左边界 |
| **X轴最大值** | 10 | 右边界 |
| **Y轴最小值** | -5 | 下边界（地下） |
| **Y轴最大值** | 20 | 上边界（高空） |
| **Z轴最小值** | -10 | 前边界 |
| **Z轴最大值** | 10 | 后边界 |

#### 典型应用场景

**场景1：防止物体无限掉落**
```
设置:
Y轴最小值: -10
Y轴最大值: 30

效果:
物体掉落到Y=-10时自动删除
保持场景清洁
```

**场景2：有限的游戏区域**
```
设置:
X: -15 到 15
Y: -5 到 20
Z: -15 到 15

效果:
创建一个30x25x30米的游戏空间
物体离开此空间即被删除
```

**场景3：太空垃圾清理**
```
设置:
X: -20 到 20
Y: -10 到 50
Z: -20 到 20

重力: Y=0（零重力）

效果:
在零重力环境下
物体漂出空间后自动删除
```

### 示例1：火山喷发模拟

**目标：** 物体从地面喷发到高空后掉落并被清理

3. 添加物体：
   添加10个球体
   设置位置: (0, 5, 0)

5. 观察：

## 🐛 常见问题

### Q1: 物体立即被删除

**原因：** 初始位置就在边界外

**解决：**
```
1. 检查物体初始位置
2. 调整边界范围
3. 或重新放置物体
```

### Q2: 看不到红色边界框

**原因：** 可视化未启用

**解决：**
```
1. 确保勾选"显示边界框"
2. 边界必须是启用状态
3. 调整相机位置查看
```

### Q3: 边界框不准确

**原因：** 边界值设置错误

**解决：**
```
检查顺序：
1. 最小值 < 最大值
2. 例如：minX=-10, maxX=10
3. 不要设置成 minX=10, maxX=-10
```

### Q4: 重力不生效

**原因：** 可能是缓存问题

**解决：**
```
1. 硬刷新页面（⌘+Shift+R）
2. 在Console中查看错误
3. 重新输入重力值
```

### Q5: 物体删除太快

**原因：** 边界太小

**解决：**
```
扩大边界范围：
X: -20 到 20（原来是 -10 到 10）
Y: -10 到 40（原来是 -5 到 20）
```

## 💡 最佳实践

### 1. 合理设置边界

**太小：** 物体很快被删除
**太大：** 失去边界意义
**推荐：** 比实际活动区域大20%

### 2. 使用可视化

**始终开启边界框显示：**
- 清楚看到活动范围
- 方便调试
- 避免意外删除

### 3. 测试重力效果

**测试步骤：**
```
1. 添加1个球体在Y=10
2. 设置不同的重力Y
3. 运行并观察下落时间
4. 找到最合适的值
```

### 4. 组合使用

**最佳组合：**
```
重力 + 边界 + 初始速度 = 完美控制

示例：
- 重力Y = -9.82（地球）
- 边界Y = -2 到 30
- 初始速度Y = 15

结果：
物体向上飞15m/s
到达最高点约11.5m
开始下落
落到底部时被删除
```
