# ✅ 重力设置简化 - 更新说明

## 🎯 改进内容

### 之前：复杂的三轴控制
```
重力 X: [0]
重力 Y: [-9.82]
重力 Z: [0]
```
❌ 界面复杂
❌ X/Z轴重力几乎用不到

### 现在：简洁的单轴控制
```
地面重力: [-9.82]

💡 推荐值: -9.82 地球 | -1.62 月球 | 0 太空
```
✅ 界面简洁
✅ 快捷按钮一键设置
✅ 专注于地面重力

---

## 🎨 新界面

### 位置
左侧面板 → 🌍 世界设置 → 地面重力

### 使用方法

**方法1：手动输入**
```
直接在输入框中输入数值
例如: -5, -3, 0, 10 等
```

**方法2：快捷按钮（推荐）**
```
点击 "地球" → 自动设置为 -9.82
点击 "月球" → 自动设置为 -1.62
点击 "太空" → 自动设置为 0
```

---

## 📊 常用重力值

| 场景 | 重力值 | 效果 |
|------|--------|------|
| 🌍 地球 | -9.82 | 正常下落 |
| 🌙 月球 | -1.62 | 缓慢下落（1/6重力） |
| 🌑 火星 | -3.71 | 较慢下落（38%重力） |
| 🌌 太空 | 0 | 零重力，物体悬浮 |
| ⬆️ 反向 | 9.82 | 物体向上"掉落" |

## 💡 使用建议

### 推荐做法
```
1. 默认使用 "地球" (-9.82)
2. 模拟太空使用 "太空" (0)
3. 特殊效果使用其他值
```

### 不推荐
```
❌ 设置正重力（物体会向上飞）
   除非特殊效果需要

❌ 极端重力值（如 -100）
   会导致模拟不稳定
```

## 📝 代码改动

**JavaScript:** 自动设置 X=0, Z=0，只控制Y轴

```javascript
// 自动设置地面重力
this.world.gravity.set(0, value, 0);
```
