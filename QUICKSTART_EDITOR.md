# 🎨 自定义碰撞编辑器 - 快速入门

## 5分钟快速上手

### 第1步：启动编辑器（1分钟）

```bash
cd /Users/williamshakespeare/Desktop/bullet-collision/web-app
npm run dev
```

然后在浏览器打开：http://localhost:5173/editor.html

### 第2步：添加刚体（1分钟）

在左侧面板点击"📦 盒子"按钮3次，添加3个盒子。

### 第3步：编辑参数（1分钟）

1. 点击选中一个盒子
2. 在右侧面板修改：
   - 质量：改为 `5` kg
   - 弹性：改为 `0.8`
   - 位置Y：改为 `8`

### 第4步：添加约束（1分钟）

1. 按住 `Ctrl` 键
2. 依次点击两个盒子
3. 选择"铰链约束"
4. 重复步骤连接第3个盒子

### 第5步：运行模拟（1分钟）

点击顶部 `▶ 运行模拟` 按钮，观察你的作品！

## 进阶操作

### 创建弹球游戏
```
1. 添加5个盒子，围成U型
2. 设置所有盒子弹性为 0.3
3. 添加1个球体，弹性设为 0.9
4. 运行模拟，用鼠标拖拽球体扔出去
```

### 创建风铃
```
1. 添加5个球体，垂直排列
2. 用点对点约束将它们连接到上方固定点
3. 设置质量为 0.1 kg（很轻）
4. 运行模拟，用鼠标轻推观察
```

### 创建小车
```
1. 添加1个扁平盒子作为车身（位置: 0, 2, 0）
2. 添加4个圆柱作为车轮
3. 用铰链约束将车轮连接到车身
4. 运行模拟，用鼠标拖拽车身移动小车
```

## 物理参数速查

| 参数 | 范围 | 效果 |
|------|------|------|
| 质量 | 0-100 | 0=静态，越大越重 |
| 摩擦力 | 0-1 | 0=滑，1=涩 |
| 弹性 | 0-1 | 0=不弹，1=超弹 |

**推荐配置：**
- 重物体：质量10kg，摩擦0.5，弹性0.2
- 轻物体：质量0.5kg，摩擦0.3，弹性0.5
- 弹性球：质量1kg，摩擦0.1，弹性0.9
- 地板：质量0，摩擦0.8，弹性0.1

## 故障排除

### 问题：刚体不动
**解决**：点击"▶ 运行模拟"按钮

### 问题：刚体穿模
**解决**：减小刚体尺寸或增加间距

### 问题：模拟太慢
**解决**：减少刚体数量（建议<50个）

### 问题：约束不工作
**解决**：
- 确保两个刚体都是动态的（质量>0）
- 检查约束类型是否合适
- 尝试重新创建约束
